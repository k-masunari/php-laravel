「MySQL 入門」等の検索ワードでWebページやYouTube動画を利用して、データベースの基本を学びましょう。以下はお勧めのリソースです。

2. Laravel環境構築

Laravelプロジェクトの立ち上げ方法は以下の通りです。 必要なツール Docker Docker Compose Laravel
手順 Laravelプロジェクトの作成 sh コードをコピーする composer create-project --prefer-dist laravel/laravel myproject cd myproject Docker Composeファイルの作成 docker-compose.ymlファイルを作成し、以下のようにMySQLサービスを追加します。 yaml コードをコピーする version: '3.8' services: mysql: image: mysql:5.7 ports: - "3306:3306" environment: MYSQL_ROOT_PASSWORD: root MYSQL_DATABASE: mydatabase MYSQL_USER: myuser MYSQL_PASSWORD: mypassword volumes: - mysql-data:/var/lib/mysql volumes: mysql-data: コンテナの起動 sh コードをコピーする docker-compose up -d

3. DBクライアントツールのインストール

Windowsの場合: HeidiSQL HeidiSQLダウンロードページからインストールファイルをダウンロードしてインストールします。 Macの場合: Sequel Ace Sequel Aceダウンロードページからアプリをダウンロードしてインストールします。

4. DBクライアントツールでMySQLに接続

接続情報の確認 Laravelプロジェクトの.envファイルに接続情報が記載されています。以下のような形式です。 makefile コードをコピーする DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=mydatabase DB_USERNAME=myuser DB_PASSWORD=mypassword DBクライアントツールで接続 HeidiSQLの場合: 新しいセッションを作成します。 「Network type」を「MySQL (TCP/IP)」に設定。 「Hostname / IP」に127.0.0.1を入力。 「User」にmyuser、「Password」にmypasswordを入力。 「Database」にmydatabaseを入力。 「Port」に3306を入力。 「Open」をクリック。 Sequel Aceの場合: 新しい接続を作成します。 「Name」に接続名を入力。 「Host」に127.0.0.1を入力。 「Username」にmyuser、「Password」にmypasswordを入力。 「Database」にmydatabaseを入力。 「Port」に3306を入力。 「Add to favorites」をクリックし、「Connect」をクリック。

5. SQL文の学習

以下の基本的なSQL文を使用して操作を行います。 テーブルの作成 sql コードをコピーする CREATE TABLE users ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(100) NOT NULL, email VARCHAR(100) NOT NULL UNIQUE, password VARCHAR(100) NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); データの挿入 sql コードをコピーする INSERT INTO users (name, email, password) VALUES ('John Doe', 'john@example.com', 'securepassword'); データの更新 sql コードをコピーする UPDATE users SET email = 'john.doe@example.com' WHERE name = 'John Doe'; データの削除 sql コードをコピーする DELETE FROM users WHERE name = 'John Doe'; これらのステップを通じて、MySQLとLaravelを利用した開発環境の立ち上げと基本的なデータベース操作を行うことができます。
